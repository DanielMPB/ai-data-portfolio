---

## 🎯 Objetivo

- Analisar dados de importação de commodities  
- Identificar países com maior volume de importação  
- Explorar padrões de mercado  
- Gerar recomendações de investimento baseadas em perfil  

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- Estruturas de dados e lógica de programação  

---

## ⚙️ Funcionalidades

### 📊 Ranking de Países
- Exibe os países com maior volume de importação  
- Permite análise comparativa entre mercados  

### 🔍 Busca por País
- Consulta detalhada de importações  
- Visualização de dados específicos  

### 📦 Análise por Commodity
- Filtragem de países por tipo de commodity  
- Organização por categorias (energia, alimentos, etc.)  

### 💡 Sistema de Recomendação
- Sugestão de commodities baseada em perfil:
  - Conservador  
  - Moderado  
  - Ousado  

---

## 📈 Possíveis Insights

- Identificação de países com alta demanda por commodities  
- Análise de oportunidades de investimento  
- Compreensão de padrões de importação
